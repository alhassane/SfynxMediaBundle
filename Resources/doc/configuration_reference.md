#Configuration Reference

All available configuration options are listed below with their default values.

``` yaml
#
# SfynxmediaBundle configuration
#       
sfynx_media:
    crop:
        formats:
            0:
                prefix: default_mosaique
                legend: Mosaique
                width: 380
                height: 380
                ratio: 1
                quality: 70
            1:
                prefix: default_big
                legend: Big
                width: 760
                height: 380
                ratio: 2
                quality: 70 
```
